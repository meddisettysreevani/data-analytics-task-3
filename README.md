# Deep-Dive Analysis & Interactive Dashboarding

## Project Overview
This project is part of the ApexPlanet Data Analytics Internship Task 3. The objective is to perform a deep-dive analysis on an e-commerce dataset and generate meaningful business insights using Python. Key Performance Indicators (KPIs) are calculated, and various visualizations are created to support business decision-making.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Google Colab / Jupyter Notebook

## Dataset
The dataset contains e-commerce sales information, including:
- Order ID
- Order Date
- Customer ID
- Customer Name
- Age
- Gender
- City
- Product
- Category
- Quantity
- Unit Price
- Total Sales

## Project Objectives
- Load and explore the dataset.
- Check for missing values.
- Perform statistical analysis.
- Calculate important business KPIs.
- Analyze sales by category, city, and product.
- Visualize sales trends using charts.

## Key Performance Indicators (KPIs)
- Total Sales
- Total Orders
- Average Order Value (AOV)
- Average Customer Age
- Top Selling Category
- Top Selling City
- Top 10 Products
- Monthly Sales Trend

## Visualizations
- Bar Chart – Sales by Category
- Pie Chart – Category-wise Sales
- Line Chart – Monthly Sales Trend
- Histogram – Customer Age Distribution
- Bar Chart – Top 10 Cities by Sales

## Key Insights
- Identified the highest-selling product categories.
- Compared sales performance across different cities.
- Analyzed monthly sales trends.
- Examined customer age distribution.
- Generated business KPIs for dashboard development.

## Conclusion
This project demonstrates how Python can be used for data analysis and visualization to uncover valuable business insights. The generated KPIs and visualizations can be integrated into an interactive dashboard using Power BI, Tableau, or Looker Studio for better decision-making.
